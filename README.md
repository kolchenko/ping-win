# 🏓 PingWin

Браузерная игра в пинг-понг с неоновым дизайном, звуком и ИИ-противником. Работает без сервера — просто открой `index.html`.

![Preview](https://img.shields.io/badge/platform-browser-blue) ![License](https://img.shields.io/badge/license-MIT-green)

## Возможности

- **Два режима:** один игрок против ИИ и два игрока на одной клавиатуре
- **Три уровня сложности:** Легко / Средне / Сложно
- **Физика мяча:** ускорение после каждых двух касаний, угол отскока зависит от точки удара
- **Звук:** фоновая чиптюн-музыка, звуки ударов, стен, голов и победы (Web Audio API — без внешних файлов)
- **Партиклы:** взрывы частиц при ударах и победе
- **Неоновый UI:** тёмная тема, глоу-эффекты, анимации очков

## Управление

| Действие | Игрок 1 | Игрок 2 |
|---|---|---|
| Вверх | `W` | `↑` |
| Вниз | `S` | `↓` |
| Пауза | `P` / `Esc` | `P` / `Esc` |

> В режиме одного игрока правую ракетку контролирует ИИ.

## Запуск

Откройте `index.html` в любом современном браузере. Никаких зависимостей не требуется.

## Технологии

- Vanilla HTML / CSS / JavaScript
- Canvas 2D API — рендеринг игры
- Web Audio API — синтез звука и музыки
